# Uber Revenue Optimization Model (Dynamic Pricing Case Study)

## Overview
This project analyzes how pricing impacts demand, revenue, and profitability using Uber as a case study. The goal is to identify how pricing strategies can be optimized to maximize revenue while maintaining demand.

## Approach
- Built a structured Excel model using weekly ride data
- Calculated key performance metrics including revenue, cost, profit, and margin
- Conducted scenario analysis to evaluate different pricing strategies
- Visualized trends in revenue and margin over time

## Key Insight
Revenue is not maximized by increasing trip volume alone. Instead, the optimal outcome occurs when pricing is balanced with demand.

## Weak Point Identified
Static pricing can leave revenue on the table during periods of high demand.

## Recommendation
Introduce dynamic pricing adjustments during high-demand periods to increase revenue while maintaining sufficient ride volume.

## Impact
- Improved revenue performance
- Higher profit margins
- More efficient demand utilization
